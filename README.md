# Spam Classifier
A Simple Spam Classifier using Logistic, SVM, Naive Bayes, Decision Tree, Random Forest, AdaBoost,  XGBoost, LightGBM, RNN, LSTM.

## Accuracies on Various Algorithms:
* Logistic Regression: 0.982353
* Support Vector Machine: 0.985882
* MultinomialNB: 0.981176
* Decision Tree Classifier: 0.962353
* Random Forest Classifier: 0.976471
* AdaBoost: 0.976471
* XGBoost: 0.976471
* LightGBM: 0.981176
* RNN: 0.94
* LSTM: 0.94
